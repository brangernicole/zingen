# Zingen

Landing page estática do **Zingen**, um app de karaokê com foco em experiência gamificada, planos de assinatura e download do aplicativo.

## 📌 Sobre o projeto

Este projeto apresenta uma página institucional/responsiva com seções de:

- Hero
- Sobre o app
- Funcionalidades
- Planos e preços
- Download
- Rodapé com navegação e links sociais

## 🛠️ Tecnologias

- HTML5
- CSS3 (arquivos organizados por seção/componente)
- Google Fonts (Inter)

## ▶️ Como executar localmente

Como é um projeto estático, você pode abrir direto no navegador:

1. Acesse a pasta do projeto.
2. Abra o arquivo `index.html` no navegador.

Ou, se preferir, use a extensão **Live Server** no VS Code para recarregamento automático.

## 🎯 Objetivo

Servir como base de estudo e prática para construção de landing pages com HTML/CSS, organização de estilos por módulos e layout responsivo.
